# Parte2_TelecomX_LATAM
Se desarrolla modelos predictivos capaces de prever qué clientes tienen mayor probabilidad de cancelar sus servicios.

El propósito central de este proyecto es crear un modelo de Machine Learning que permita anticipar la cancelación o abandono de clientes. A través del estudio de variables relacionadas con las características demográficas de los usuarios y los servicios que utilizan, se identifican patrones y factores clave asociados al churn. Esta información permite a la empresa implementar acciones de retención de manera anticipada, con el fin de intervenir antes de que el cliente decida finalizar su relación con el servicio.
